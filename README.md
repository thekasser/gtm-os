# gtm-os

GTM Operating System — modular, spec-driven services covering planning through revenue recognition.

**Naming convention**: the 40 components in L1–L8 are **GTM Services** — deterministic, single-writer-per-table, audit-grade. The term **Agents** is reserved for the forthcoming reasoning layer (Tier 2) that reads from services and proposes plays for human review without writing canonical data. The distinction is deliberate; see the architecture evaluation for context.
